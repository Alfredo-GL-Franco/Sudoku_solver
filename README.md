# Sudoku_solver
Este proyecto consiste en un solucionador de Sudoku desarrollado en C++ utilizando el algoritmo de backtracking. El programa es capaz de resolver cualquier tablero de Sudoku válido, ya que evalúa de manera precisa si un número puede colocarse en una casilla respetando las reglas fundamentales del juego: que no se repita en su fila, columna ni en el cuadrante 3x3 correspondiente. El proceso de resolución se basa en la búsqueda recursiva de soluciones, intentando números del 1 al 9 en cada espacio vacío y retrocediendo cuando una elección no conduce a una solución válida. Esto convierte al programa en un ejemplo ideal del uso de recursividad y técnicas de exploración exhaustiva aplicadas en la práctica.

El código está completamente documentado con Doxygen, lo que facilita su comprensión, mantenimiento y ampliación. La estructura del proyecto incluye un archivo principal encargado de ejecutar el programa, así como archivos separados para la implementación y la declaración de funciones, favoreciendo un diseño modular y ordenado. Asimismo, se incorpora un tablero de Sudoku predefinido que permite observar claramente el funcionamiento del algoritmo antes y después del proceso de resolución.

El proyecto puede compilarse con cualquier compilador moderno de C++ y no requiere dependencias externas. Además, la documentación puede generarse fácilmente mediante Doxygen para obtener una referencia visual y detallada de cada función. Este proyecto es ideal tanto para estudiantes como para desarrolladores que deseen estudiar o ejemplificar el funcionamiento de algoritmos de backtracking, la manipulación de matrices, la validación de reglas lógicas y la implementación limpia y modular de un solucionador de problemas en C++.
-------------------------------------------------------------------------------------------------------------------------------------------
This project is a Sudoku solver developed in C++ using the backtracking algorithm. The program is capable of solving any valid Sudoku board by carefully evaluating whether a number can be placed in a specific cell while respecting the key rules of the game: the number must not appear in the same row, column, or 3×3 subgrid. The solving process is based on recursive search, trying numbers from 1 to 9 in each empty cell and backtracking whenever a chosen value leads to an invalid or unsolvable state. This makes the project an excellent practical example of recursion and exhaustive search techniques applied to a real problem.

The code is fully documented using Doxygen, making it easy to understand, maintain, and extend. The project is structured into separate files for function declarations, implementations, and the main program, providing a clean and modular organization. A predefined Sudoku board is included to clearly demonstrate how the algorithm works, showing both the initial puzzle and its final solved state.

The project compiles with any modern C++ compiler and requires no external libraries. Documentation can be generated through Doxygen to produce a detailed visual reference of all functions and internal logic. This project is ideal for students, educators, and developers who want to study or demonstrate how backtracking works, how to handle matrix-based logic, and how to implement a clear and modular problem-solving program in C++.
